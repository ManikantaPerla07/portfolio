## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ManikantaPerla07/portfolio.git
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
